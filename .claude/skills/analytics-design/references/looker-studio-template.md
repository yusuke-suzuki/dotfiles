## グラフタイプの選択

| 目的 | グラフタイプ |
|-----|------------|
| KPI の現在値 | スコアカード |
| 時系列トレンド | 期間グラフ (時系列) |
| カテゴリ別の時系列 | 積み上げ面グラフ / 積み上げ棒グラフ |
| カテゴリ比較 | 棒グラフ |
| 構成比 | 円グラフ |
| 詳細データ | テーブル |
| 分布 (パーセンタイル) | 期間グラフ (複数指標) |

## 設定項目の説明

- **ディメンション**: グループ化の軸（日付、カテゴリなど）
- **内訳ディメンション**: さらに細かく分類するための軸
- **期間のディメンション**: 期間フィルタが適用されるフィールド
- **指標**: 集計方法（Record Count, SUM, AVG, PERCENTILE など）
- **フィルタ**: 表示データの絞り込み条件
