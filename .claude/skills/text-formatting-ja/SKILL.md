# Japanese Text Formatting

Apply these rules to prose text (documents, comments, descriptions).

## Spacing

Add half-width spaces around alphanumeric characters and English words:

- OK: `追加された権限 (18 件)`
- NG: `追加された権限（18件）`

## Parentheses

Use half-width `()` instead of full-width `（）`:

- OK: `ユーザー (User) が追加されました`
- NG: `ユーザー（User）が追加されました`

## Important

Do NOT modify actual data values. Preserve text in backticks, database values, and API responses as-is. These rules apply only to human-readable prose.
